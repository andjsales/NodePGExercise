# Node PG Exercise

## Description

- A REST-ful backend API server for a simple company/invoice tracker
- Node.js and Express

## Setup

- express and pg via NPM

## How to use?
### Companies
Starting the server:
- npm start
GET /companies
- list all companies
GET /companies/[code]
POST /companies
PUT /companies/[code]
DELETE /companies/[code]
